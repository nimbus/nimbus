Neovex demos

Browser demos for a reactive API + WebSocket backend.

Neovex keeps native demos and Convex-surface demos side by side here. The native demos run directly against Neovex routes, while the Convex demos exercise the wrapper packages and Convex-style runtime surface.

neovex/html

A single-page playground using neovex/rest SDK for creating a tenant, installing a schema, inserting documents, scheduling a mutation, and watching live subscription results.

Open demo
Run npm run neovex:demo:html (Vite dev server on port 5177).

firebase/html

Browser demo using @neovex/firebase over REST or gRPC-Web unary calls plus the browser WebSocket Listen bridge.

Open demo
Run npm run firebase:server:html and npm run firebase:demo:html.

convex/html

React demo using convex/react and generated refs over the Convex wrapper surface.

Run npm run convex:server:html and npm run convex:demo:html.

convex/http

Browser HTTP demo using convex/browser, generated refs, and Convex-style actions and queries.

Run npm run convex:server:http and npm run convex:demo:http.

mongodb/node

Node.js demo using @neovex/mongodb over the MongoDB wire protocol for CRUD and aggregation.

Run npm run demo --workspace mongodb-node.
